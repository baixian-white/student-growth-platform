# 全国青少年科技教育成果展示大赛

**年度**：第五届（2024-2025学年）

## 赛事介绍
为认真贯彻习近平总书记关于教育强国、科技强国、人才强国建设的重要论述，落实党中央关于实施科教兴国战略的重大部署，经教育部批准 (教监管厅函〔2022〕13号)，由中国下一代教育基金会在2022-2025学年每年主办一届全国青少年科技教育成果展示大赛，简称“全国青科赛”。

大赛以“科技强国·未来有我”为主题，以“坚持立德树人根本任务，培养具有家国情怀、崇尚科学的新时代好少年”为目标，彰显公益普惠风尚，助力全面实施素质教育，为建设高质量教育体系贡献力量。大赛注重发挥公益慈善组织在培育科学精神、培养创新人才、营造创新氛围等方面的作用，组织开展适合青少年特点的展示和竞赛活动。

## 主办单位
**批准单位**：教育部
**指导单位**：中国关心下一代工作委员会、教育部关心下一代工作委员会
**主办单位**：中国下一代教育基金会

## 参赛要求
**参赛学段**：小学、初中、高中、中专、职高在校就读学生
竞赛实行“自下而上、逐级选拔”，分为省级区域赛和全国总决赛。

## 赛项设置
大赛围绕青少年信息素养、人工智能与机器人创新应用等领域，设立以下三大主要赛项类别：
1. **人工智能竞技类**
2. **科技创意设计类**
3. **仿真创新应用类**

## 赛程安排（以各省市承办单位公布为准）
各地报名时间大多在 **2025年4月至5月中旬** 截止。部分地区时间安排参考如下：
- **北京赛区**：报名截止 2025年4月27日；比赛时间 5月24日-25日
- **天津赛区**：报名截止 2025年5月15日；比赛时间 5月24日-25日
- **上海赛区**：报名截止 2025年5月15日；比赛时间 5月24日-25日、6月1日
- **线上赛项**：大部分省级赛区的线上赛集中在 **2025年5月24日-25日** 进行。

## 奖项与证书设置
大赛坚持公开、公平、公正的评审原则进行评选，设立如下奖项：
1. **省级区域赛等级奖**：为在省级选拔赛中表现优异的学生颁发。
2. **全国总决赛等级奖**：设置金奖、银奖、铜奖等。
3. **指导教师证书**：对在大赛中组织出色、指导有方、学生成绩优异的指导教师颁发认证证书。

## 大赛其他规定
- 大赛坚持公益属性，所有捐赠、赞助资金和物资，均统一进入中国下一代教育基金会指定账户管理核算，中国下一代教育基金会不委托任何机构和个人代收款物。
- 参赛对象须经学校推荐或在符合条件的平台通过合规路径申报，积极遵循各项章程和评审规则。
