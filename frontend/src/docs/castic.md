# 青少年科技创新大赛

**时间**：2022-09-06 14:48:56

## 赛事介绍:
（白名单赛事之一）北京青少年科技创新大赛是一项青少年科技创新成果和科学探究项目的综合性科技竞赛，是目前面向在校中小学生开展的规模最大、层次最高、具有示范性和导向性的科技教育活动。这是全市各个区每年重要的赛事之一，获奖成绩也备受各重点学校老师重视，是北京各大高校《科技特长生简章》中明确认可的赛事之一。

## 主办单位:
教育委员会、科学技术协会、科学和信息化委员会

## 项目介绍:
创新大赛分为青少年和科技辅导员两个板块，包括小学生科技创新成果竞赛、中学生科技创新成果竞赛、少年儿童科学幻想绘画比赛、科技辅导员科技创新成果竞赛、青少年科技实践活动比赛。

## 赛程安排：区赛-市赛-国赛 (2021-2022年为例)
- 区级比赛报名截止：11月-次年1月
- 市级比赛报名截止：次年12月-次年2月
- 全国赛报名截止：次年3月-次年5月

**2021年-2022年的具体安排及流程**：
- 1月28日-2月10日 网上申报、网上素质测评、提交签字盖章版申报材料
- 3月初 初审结果公示，终评项目在线展示咨询
- 3月中旬 终评学生布展、展前复审、专家评审、学生终评展示、公众开放日
- 3月26日 北京市青少年科技创新大赛终评获奖名单公示

## 评奖办法:
（一）区青少年科技创新大赛是在各学校组织竞赛活动的基础上举办的，参赛者要首先参加学校评选活动，经过评选可以被推荐参加区级初评。
（二）进入区级终评的作品，按要求参加现场答辩，由专家评定出成绩并择优推荐参加北京青少年科技创新大赛。
1、学生奖项：根据评比成绩评选一、二、三等奖若干名并颁发获奖证书。
2、教师奖项：根据教师指导学生的获奖情况，评出优秀辅导教师奖若干名并颁发证书。
3、优秀组织奖：赛事组织部门将根据各学校的活动参与情况以及参赛质量评定出优秀组织奖若干名并颁发获奖证书。

## 报名时间、地点、方法 (以北京海淀为例)
（一）所有参赛人员应以校级为单位申报，不接受作者个人直接申报。
（二）请各校于1月28日-2月10日将参赛材料上传；
（三）参赛材料要求 **(本届比赛只接收电子版申报！！！)**

## 参赛要求

### 1. 青少年科技创新成果竞赛 (小学技术、中学工程、科学研究论文类项目)
**申报材料**
（1）项目申报书、研究论文或研究报告，报告数字应在一万字以内
（2）作品的原始实验数据记录、研究日志等 (电子版)
（3）文件电子版总大小不超过5MB。 请严格遵守此条数

### 2. 科幻画作品申报材料
（1） 一律规格为4开的纸质或是其它材料。作品要求干净、整洁
（2） 艺术形式包括： 油画、国画、水彩画、水粉画、钢笔画、铅笔画、蜡笔画、版画、粘贴画、电脑绘画。绘画风格及使用材料不限，但不包括非绘画类的其它美术品与工艺品
（3）参赛作品限个人申报， 即由作者本人独立完成的作品。不接受集体作品参赛。参赛作品不得 抄袭他人作品，违者一经发现，将被取消参赛资格

### 3. 青少年科技实践活动申报材料
①项目申报书一份。
②活动报告一份：字数1万字以内。
③活动附件：大小不超过5MB。
④青少年科技实践活动各校限报一项
⑤项目申报书中“申报者确认事宜”项，按照要求签字后上传扫描件，放置在申报者文件夹中。
⑥学校参加青少年科技实践活动项目汇总表（附件4）。

### 4. 科技辅导员科技创新成果竞赛申报材料
①项目申报书一份。
②项目报告一份。
③其它附件（照片或视频）。
④暂不交实物。
⑤项目申报书中“申报者确认事宜”项，按照要求签字后上传扫描件，放置在申报者文件夹中。
⑥学校参加科技辅导员创新成果竞赛项目汇总表（附件5）。

### 5. 青少年科技创新成果竞赛 (技术、工程类) 项目:
1. 答辩形式：现场集中问辩 (分为小学组、初中组、高中组)。
2. 答辩时间、地点：待定，届时提前通知。
3. 答辩需要：参赛选手提前准备项目展板，答辩当天带到比赛场地 (大赛提供展架)。
展板规格90×120cm。参展实物高不超过 2 米,宽不超过 1.5 米,重量不超过 100 千克。项目展示材料中不能有易燃、易爆危险品和管制刀具。展品用电电压不得超过220 伏。
