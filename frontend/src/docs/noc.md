# 全国中小学信息技术创新与实践大赛 (NOC)

**官方网站**：[s.noc.net.cn](http://s.noc.net.cn/)

## 大赛简介
全国中小学信息技术创新与实践大赛（简称 NOC 大赛，NOC 为 Novelty, Originality, Creativity 的缩写）是一项旨在全面普及信息技术教育，弘扬创新精神、培养实践能力，面向广大中小学生开展的计算机科学与技术及学术性综合实践活动。

NOC大赛自2002年在人民大会堂启动以来，已连续举办多届（已至第二十余届）。它是**教育部批准的面向中小学生的全国性竞赛活动（白名单赛事）**之一，在国内基础教育领域和信息技术教育领域具有深远影响。

## 赛项与内容设置
NOC大赛紧跟科技发展前沿，涵盖极其丰富的项目类别，通常包含以下几大核心方向：
- **编程与算法类**：如 C++ 编程、Python 编程、Scratch/图形化创意编程等，注重考察学生的计算思维、逻辑推理与代码实现能力。
- **人工智能与机器人创意类**：包括智能制造赛道、开源硬件创意设计、各类智能机器人任务挑战赛、无人机编程飞行等。
- **数字艺术与创客创作类**：如 3D 打印创意设计、微视频制作、数字计算与艺术绘画等，强调科技与人文艺术的跨界融合。

## 赛制与赛程安排
大赛实行层层选拔机制，分为 **地方赛（省级/市级赛）** 和 **全国总决赛** 两大阶段：
1. **组织报名与地方赛选拔**：每学年的上学期至下学期初期启动，报名与地方赛选拔周期多集中在每年的 **3月至5月**。
2. **全国总决赛**：经过地方赛的重重选拔，晋级选手将在每年的 **7月至8月（暑期）** 参加全国总决赛，角逐最高荣誉。

*(注：受各项客观因素影响，部分年度、部分赛项可能会安排“线上赛”+“线下赛”的形式灵活开展。具体日程以当年度NOC大赛官方最新发布的通知公告为准。)*

## 参赛对象
广泛面向全国（含港澳台地区）在校的**小学、初中、高中（含中职）学生**。根据不同赛道的技术深度要求，细分为：
- 小学低年级组 (1-3年级)
- 小学高年级组 (4-6年级)
- 初中组
- 高中组

## 奖项评估与激励
- **学生奖项**：全国总决赛按比例设立一等奖、二等奖、三等奖，由主办单位颁发荣誉证书。部分表现极其卓越、具备突出创新发明潜质的选手，还有机会角逐特设的 **“恩欧希教育信息化发明创新奖”**。
- **教师及单位奖项**：向辛勤辅导的优秀指导教师颁发证书，并对在赛事组织和推广中表现突出的省市教育机构和学校颁发“优秀组织奖”。
